Go to: https://github.com/homekrypto/REH-all-working-with-amazon-database/settings/secrets/actions

For each secret below, click "New repository secret" and add the corresponding value
from your amplify-env-vars-private.txt file:

REQUIRED SECRETS:
================

1. DATABASE_URL - Your PostgreSQL connection string
2. NODE_ENV - Set to: production
3. NEXTAUTH_SECRET - Your NextAuth secret key
4. GOOGLE_CLIENT_ID - Your Google OAuth client ID
5. GOOGLE_CLIENT_SECRET - Your Google OAuth client secret
6. STRIPE_PUBLISHABLE_KEY - Your Stripe publishable key (pk_test_...)
7. STRIPE_SECRET_KEY - Your Stripe secret key (sk_test_...)
8. STRIPE_WEBHOOK_SECRET - Your Stripe webhook secret (whsec_...)
9. AWS_ACCESS_KEY_ID - Your AWS access key ID (AKIA...)
10. AWS_SECRET_ACCESS_KEY - Your AWS secret access key
11. AWS_REGION - Set to: eu-north-1
12. AWS_S3_BUCKET_NAME - Your S3 bucket name
13. SMTP_HOST - Set to: smtp.gmail.com
14. SMTP_PORT - Set to: 587
15. SMTP_SECURE - Set to: false
16. SMTP_USER - Your email address
17. SMTP_PASS - Your email app password
18. EMAIL_FROM - Set to: Real Estate Platform <noreply@realestateplatform.com>

OPTIONAL (add after first deployment):
======================================
19. NEXTAUTH_URL - Your deployed Amplify URL
20. AMPLIFY_APP_ID - Amplify app ID (if you have one)

üìù Get actual values from: amplify-env-vars-private.txt (not committed to repo)

After adding all secrets, the GitHub Actions workflow will automatically 
deploy when you push commits to the main branch!
